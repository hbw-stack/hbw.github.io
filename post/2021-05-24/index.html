2021-05-25 1 min read # pyqt # bug # 日记

编译了一个PyQt5，反正放到arm上就是很多要编译的。编译PyQt5的要求：qmake、sip，注意sip需要安装旧的4.19的，这样才有可执行的sip文件。主要参考：https://forums.developer.nvidia.com/t/how-to-install-pyqt5-in-jetson-xavier/79554。最后的bug是QtWidgets.sip not found，这是一个包没有，使用pip安装即可（pip install PyQt5.sip）。在编译的时候，我还学到了make -j4是使用四个线程同时编译的意思，由于这个jetson有8个核，所以最大可以设置到8，编译速度非常快。

论文查重：paperpass 13%，知网 15.7%，主要还是前面第一章复制的比例太高了，而且自己还没有做降重，所以知网的查重比非常高。
